# Project Summary
**Manejo Certo** is an innovative financial management application specifically designed for livestock farming. It provides farmers with essential tools to efficiently manage transactions, lots, and sales, thereby enhancing productivity and profitability. By utilizing modern technologies, the application aims to streamline financial operations, contributing to sustainable livestock management practices.

# Project Module Description
The project consists of several key modules:
1. **Backend Module**: Manages data storage and API interactions via Supabase, including database setup and business logic.
2. **Frontend Module**: Offers a responsive user interface built with React and TypeScript, facilitating effective interaction with the financial management system.
3. **Zootecnical Parameters Module**: Enables users to register and edit zootecnical metrics for various management phases.
4. **Viability Simulator Module**: Allows users to simulate cattle purchases based on input parameters and existing zootecnical data.

# File Description Inventory
- **README.md**: Instructions for setting up the backend and frontend environments.
- **skills_docs/**: Documentation related to Supabase integration and edge functions.
- **components.json**: Lists UI components used in the frontend.
- **public/**: Contains static assets such as images and favicons.
- **src/**: Main source code for the application, including components, styles, and hooks.
- **seo-scripts/**: Scripts for generating SEO-related files, such as sitemaps.
- **pages/CompraVenda.tsx**: Manages the purchase and sale processes, including lot creation and transaction recording.
- **pages/Dashboard.tsx**: Displays financial summaries and analytics for users.
- **pages/LoteDetalhe.tsx**: Provides detailed insights into specific lots, including financial metrics and transaction history.
- **pages/Lotes.tsx**: Displays and manages active and historical lots.
- **pages/NovoLancamento.tsx**: Facilitates the creation of new financial transactions.
- **pages/Parametros.tsx**: Manages zootecnical parameters for different management phases.
- **pages/Simulador.tsx**: Simulates cattle purchases based on input parameters and pre-filled zootecnical data.
- **lib/supabase.ts**: Configures the Supabase client and contains utility functions for data manipulation.
- **lib/types.ts**: Type definitions for various entities used in the application.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Supabase, Node.js

# Usage
To install dependencies, run:
```bash
pnpm install
```
To build the project, use:
```bash
pnpm run build
```
To run the application, execute:
```bash
pnpm start
```
